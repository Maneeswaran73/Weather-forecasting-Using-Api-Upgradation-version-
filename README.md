# Weather-forecasting-Using-Api-Upgradation-version-
The Weather Forecasting Web Application is a simple and responsive web-based project that allows users to check real-time weather information for any city across the world. The app fetches data from a public weather API and displays details such as temperature, humidity, wind speed, and weather conditions in an interactive interface.
